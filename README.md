# miniguia-notebooklm
Caderno NotebookLM para apoio de novos investidores correntistas do do Bradesco (Desafio DIO)

🎯 **Contexto e Objetivos**
Assunto escolhido: **Guia Completo de Investimentos e Serviços Bradesco**
Objetivos de estudo: 
- criar um assistente de conhecimento "Segundo Cérebro" para correntistas do Bradesco, leigos em investimentos;
- fornecer fontes com dicas para investimentos atgravés do Banco Bradesco; e     
- gerar materiais rápidos ou densos para estudo e comparação dos divervos tipos de investimentos disponíveis no Banco Bradesco.
